Hello! Project Kids

Hello! Project Kids是由日本女子偶像团体Hello! Project举办的Hello! Project Kids Audition中选拔出来的全部15人的总称。2004年1月其中8人组成了Berryz工房，2005年6月剩余7人组成了℃-ute。

Hello! Project Kids Audition是应募对象限定为小学生的Hello! Project选秀会，2002年6月30日在东京电视台的「Hello! Morning」节目中宣布了合格者。最终剩下15人、她们被称之为Hello! Project Kids（2004年举办了类似的选秀会Hello! Project EGG、这批合格者也就被称之为Hello! Project EGG了）。合格当时15人全部是小学生，但伴随著年龄的增长到2008年4月为止所有人都是中学生，2011年4月为止所有人都已是高中生。

2005年石村舞波从Berryz工房毕业，2006年1月有原栞菜加入℃-ute，同年10月村上爱退出℃-ute，2009年7月有原栞菜退出℃-ute，同年10月梅田绘理香从℃-ute毕业，目前Berryz工房和℃-ute各是7人和5人。
