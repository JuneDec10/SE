Windows XP Professional x64 Edition

Windows XP Professional x64 Edition是微软Windows XP作业系统专业版之一，支援AMD64架构，包括AMD64及EM64T处理机。2005年4月25日推出，起初是对AMD64系统平台的回应做为安腾版的补充，后随安腾版的夭折成为XP中最终唯一受支援的64位元平台产品。现阶段该系统己停止以任何形式发售，并已于2014年4月8日停止所有支援。

原生版本仅供日本语版和美国英语版。

美国英语版可附加界面语言：日本语、西班牙语、法语、义大利语、瑞典语、德语、韩国朝鲜语、中文(简体)、中文(繁体)

Windows XP Professional x64主要以OEM或COEM方式提供给个体使用者，企业与组织以Volume Licence形式获得。

与Windows Server 2003保持一致，最终级别在Service Pack 2阶段。

2005年4月， 原始版本发布，级别Service Pack 1，原生版语言美国英语、日本语。

2006年5月， 第一补丁原生版，级别Service Pack 1b， 原生版语言美国英语、日本语。

2007年3月， 第二修正版，级别Service Pack 2, 原生版语言美国英语。

WoW64的技术容许大多数的32位元应用程式在性能不受损的情况下运行。

WoW32未支援，因此无法运转16位元的视窗及MS-DOS应用程式。 但为了兼容使用16位元安装程式的纯32位元软体的遗产保护，部分的16位元安装程式被提供支援。


仅供支援AMD64或Intel64技术的电脑使用，不支援基于安腾64位元的系统，不支援AHCI和USB 3.0设备。

最大支援128GB物理记忆体。

最大支援两颗物理处理机芯片以及64个逻辑处理单元核心。

磁碟机：支援最大为2TB的GPT型单一磁碟分割域。

不支援微软Office 2010及之后的版本。

Windows XP Professional x64 Edition实际上是Windows XP家族中专业版的64位元产品，并不是单独的64位元版。
