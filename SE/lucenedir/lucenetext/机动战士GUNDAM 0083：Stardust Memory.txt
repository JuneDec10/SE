机动战士GUNDAM 0083：Stardust Memory

《机动战士GUNDAM0083：STARDUST MEMORY》（，"）是GUNDAM系列作品当中的一部OVA动画。一共13集、1991年5月份最初在日本发行、最后一集在1992年9月份发行完毕。

一年战争完结后三年，基于地球联邦军重建计划而被提议的「GUNDAM开发计划」开始执行，其中两台原型机RX-78GP01与RX-78GP02A为了进行性能测试而搬运至澳大利亚"-{zh-hans:托灵顿; zh-hant:特林顿;}-"基地。吉翁残党查知此项计划，于是袭击基地企图夺取装载有核弹的GP02A。曾经被称为「所罗门的恶梦」的吉翁军王牌-{zh-hans:机师; zh-hant:驾驶员;}-阿纳贝尔·卡多少校成功的搭上了GP02A，但是在企图脱离的GP02A面前，新手测试驾驶员浦木宏驾驶了GP01拦阻，揭开了宇宙世纪0083年迪拉兹纷争的序幕。

夺取GP02的阿纳贝尔·卡多前往非洲金巴莱德矿山基地，在当地残党军领袖-{zh-hans:诺渊·比达; zh-hant:诺恩·比达;}-少将的协助下成功以HLV脱离地球，回到暗礁空域，由埃丘·迪拉兹中将所领导的吉翁残党（迪拉兹舰队）基地。并纠集地球圈吉翁军残存势力向地球联邦宣战，作战代号为「星尘」。其后阿纳贝尔·卡多以GP02A袭击联邦军于金米岛（旧吉翁军所罗门要塞）宙域举行的阅舰典礼，并以核弹摧毁了2/3的联邦舰队，战斗中GP01Fb与GP02同归于尽。同时西玛·卡拉豪中校所率领的海军陆战部队，劫持两座太空殖民地，并使其中之一向月球落下，而后借由月面都市冯·布朗的设施令其再改变轨道朝向地球而去。

正当迪拉兹舰队为使殖民地顺利落下，准备与联邦军轨道机动舰队决战时，西玛在联邦军的许诺下叛变并杀害迪拉兹(实为迪拉兹被西玛胁持当人质时自尽)。联邦军方的加米托夫派也试图以SOLAR SYSTEM II系统烧毁殖民地。加米托夫派试图把自身塑造成一个最后拯救地球的英雄形象，待吉翁宣布投降后，打击联邦内部崛起的高文派成员。卡多则趁着高文派对已投向加米托夫派的西玛舰队攻击所造成的阵线混乱，率领残余部队突破联邦军所布置的SOLAR SYSTEM II防线并成功摧毁天幕II系统控制舰。在UC0083年11月13日凌晨00:34分、殖民地最终撞入地球、但是没有像联邦军预期的那样撞到总部-{zh-hans:加布罗;zh-hk:贾布罗;zh-tw:贾布罗;}-，而是落在北美洲平原谷仓地带。殖民地下落后加米托夫派则改变计划利用新的恐慌建立了-{zh-hans:泰坦斯;zh-hk:泰坦斯;zh-tw:迪坦斯;}-，控制了军队。

本作品如同其标题"0083"主要是叙述机动战士GUNDAM（0079）到机动战士Z GUNDAM（0087）之间所发生的故事，为历史的空白作了填补。0083本身在影像质量上就是非常高的作品，由于在系列发售中途就决定借由再编集的方式制作剧场版，所以系列后半的作画质量又更加提升。

在作品当中以个人观点来看待这场战争的妮娜·帕布尔顿与憎恨所谓「吉翁大义」的西玛·卡拉豪等人，对于身为「敌人」角色的迪拉兹阵营多所批判。但是在0083的爱好者当中认同迪拉兹思想的比例却占了压倒性的多数。一般认为这是对于迪拉兹本身人格的高洁以及卡多对于其敌手浦木宏的应对博得了纯粹的好感所致。其后所制作的剧场版「吉翁的残光」更删去了大部分与妮娜有关的戏份，使得整部作品看来更像是迪拉兹舰队的赞美诗。

联邦军队内部的派系倾轧在整个星尘作战中起到了巨大的作用，首先在明知对方控制核武的情况下，仍然执行舰队检阅，以诱惑迪拉兹舰队决战。之后在有可能阻止殖民地落入地球的情况下，由于阿鲁比翁号舰长派系不服从加米托夫派巴斯古的指挥造成防守力不足以及浦木宏等人执意对友军西玛舰队的攻击造成的混乱，给了卡多的特攻的机会。最后卡多成功击毁「天幕II」系统控制舰，使「天幕II」未能完全消灭殖民地以阻止其坠落地球，失去了最后挽回的机会。但同时也借此打压了联邦军内部的高文派，巩固了其在联邦军内部的地位。

同时商人与投机主义者的战争时行为也值得深思。商人亚纳海姆电子冯·布朗基地的负责人，在战争中微妙的保持双方的力量平衡以从中牟利。投机主义者西玛·卡拉豪在战争中完全处于自己的立场考虑，两面讨好以谋求对自己有利的局面。

本来0083的制作计划可说是从Gundam Sentinel的动画化企画所开始，所以实际的角色，设定资料，与故事走向与GUNDAM Sentinel有许多类似之处。经过GUNDAM Sentinel与0083的洗礼，一部份著重军武设定与描写的「军事派」钢弹迷抬头，并且经常以严厉的角度来检视后续GUNDAM作品的写实性。

（吉翁的残光）于1992年上映。
