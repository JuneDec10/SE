Microsoft FrontPage

Microsoft FrontPage（全名：Microsoft Office FrontPage，简称FP）是微软公司推出的一款网页设计软件。它于1997年至2003年的时候属于Microsoft Office系列的其中一款产品。FrontPage由于其易用性，被认为适合网页初学者的制作工具。但因其功能无法满足更高要求，自2006年，Frontpage已经被Microsoft Expression Web及Microsoft SharePoint Designer取代。

无论要建立个人网页、资讯网页，还是公司的互联网或内联网网站，Microsoft FrontPage都能够提供一个建构与管理网站所需的工具。由于它的操作方式就和其它Microsoft Office办公室软件相似，所以FrontPage是十分容易去掌握及使用。



Microsoft FrontPage的最后一个版本是FrontPage 2003，不再推出新的版本，并由Microsoft SharePoint Designer与Microsoft Expression Web完全取代。

2006年，微软公司宣布Microsoft FrontPage将会被Microsoft SharePoint Designer与Microsoft Expression Web新产品替代。这两款正在开发中的软件都是部分基于Microsoft FrontPage的。

Microsoft Office System 2007已经包含Microsoft SharePoint Designer。

Microsoft Expression Studio则包含Microsoft Expression Web

由于FrontPage的易用性和所见即所得，它曾是中国大陆中小学信息技术课程的必备软件。
