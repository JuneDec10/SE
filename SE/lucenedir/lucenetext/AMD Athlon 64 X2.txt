AMD Athlon 64 X2

Athlon 64 X2是AMD设计的首款桌面级双核心处理器，脚位有Socket 939、AM2，于2005年5月首次推出。首批产品采用90奈米SOI制程，其后也开始推出代号 Brisbane 的65奈米产品，并于2006年12月发售，全数产品均支援SSE3指令集。处理器内建两颗Athlon 64核心，两个核心可以透过 System Request Queue 互相沟通。Intel Pentium D的两个核心沟通必须透过FSB，绕过北桥方能；换言之，这会增加FSB的负担，减低系统性能。相比之下，由于Athlon 64 X2有专属通道，系统性能比Pentium D高，但售价较贵。

与单核心的Athlon64比较，这款双核产品拥有更多的晶体管，因此晶圆需使用更复杂的制程，期间必须有更大面积的矽为无损，使其产量不及单核产品，因此售价也会比单核产品昂贵。

由于Athlon X2对外连接是HyperTransport，与Athlon 64一样；所以使用脚位为Socket 939或SocketAM2的单核心Athlon 64的用户，只要更新主机板BIOS，即可直接升级为Athlon 64 X2。这一点比Intel优胜。

早期的Athlon X2处理器是有2 x 512 KB及2 x 1024 KB两种L2快取记忆体版本的，及至2006年中AMD宣布停产2 x 1024 KB L2的Athlon X2。

另有更低阶的X2 3600+型号，时脉与3800+相同，但L2被减为2 x 256 KB。这款产品最早是提供OEM厂商使用的，后来为应对Intel Core 2上市，AMD也于2006年7月至8月发行AM2版本的3600+，与Intel的Pentium D 915/925竞争。

AMD于2006年12月发售65nm产品，代号Brisbane，产品仍旧使用K8微架构，而使用K10微架构的新产品预计会于2007年中推出。另外在2007年6月推出的Brisbane 45W BE省电版本的双核处理器中，新品改称为「Athlon X2」，去掉 "64" 字样，并使用全新型号，原有的4000+及4400+另外推出型号为BE-2350及BE-2400省电型处理器。

部份双核产品型号属Black Edition黑盒版本，它们与普通版的分别在于不锁倍频，用家可自由调整倍频，发挥其极限性能。现有的Athlon X2系列的Black Edition型号有90nm F3步进值的6400+处理器及65nm G2步进值的5400+、5000+处理器。

2006年6月中，AMD表示将不会继续出产每核1MB L2 Cache的Athlon 64及Athlon 64 X2处理器，并集中在Opteron及较高型号的产品上，因此仅出产了少量2 x 1MB L2的Athlon 64 X2 4000+、4400+及4800+。
