牌九

推牌九，是中国传统骨牌游戏，玩法是依据两扇骨牌点数的不同组合，来比较大小，以决定胜负。据说牌九始于宋代。

明清时主要以牌九来赌博，因此中共建政后，严格取缔。

牌九，即排出九的数目之意。牌九最大的对牌为“至尊宝”。

三十二只骨牌 2～3～4颗骰子（注：若3颗骰子则颜色相同，4颗则其中1颗颜色不同，3颗或4颗依赌场而异。）；1个骰盅。

在32只牌当中，有11种牌有两只.这种成双的牌称"文牌" 或"文子"。
在32只牌当中，有10只牌没有成双。其中8只有点数相同，但图案不同的「对应牌」。而2种只有单独一只。这种牌共有10只，称"武牌" 或"武子"。

骨牌牌九的基本玩法就是以骨牌点数大小分胜负。骨牌牌九又分大牌九与小牌九，大牌九是每人四张牌，分为大小两组，分别与庄家比牌，全胜全败为胜负，一胜一败为和局；小牌九是每人两张牌，胜负立现，由于干脆利落，小牌九于民间流行较广，但正规赌场仍以大牌九为主。

主要与庄家比大小，特别的是玩家可作庄与其他玩家对赌，也可轮流做庄。人数包括庄家通常是4人或8人。

各人下注后，由庄家将所有牌面朝下，开始砌牌，然后以8排每排4张排列。用骰子掷出点数，然后按顺序将牌分配到每个参与者手中。

分配牌的方法各赌场有异，以下是一种分牌顺序。


玩家会有4只牌，分开两组，每组两张。玩家可自行将四牌两两搭配，然后两组牌朝下，小的点数横摆放在前面，大的点数直摆放在后面(由于过于繁复已少人用，现在多不分大小横直放)。然后每人与庄家比牌分胜负，必须前后都大于对方才算赢，前赢后输或前输后赢就是和局，前后都输即输。所以配牌必须讲究策略。

牌九另一种较灵活的赌法是可以轮流做庄，通常是由赌场做庄开始，然后依逆时针方向轮流做庄，玩家可以独自做庄或与赌场合伙做庄。

对牌或称对子，是成对的二牌。以下列的是所有对牌大小顺序。但算不算对牌因赌场而异。例如，天王、地王、天杠、地杠、天高九及地高九等，有些赌场并不列为对牌，尤其是天高九及地高九。

若没有以上的对牌，则以二牌之和的个位数分胜负.最大是9,最小是0。这也是牌九的名称由来，即排出九的数目之意。

例: 以两只牌组合为例
1. A.一鹅牌/和牌(4点)及 B.一杂五(5点)的组合，二牌点数之和为4+5=9，个位数为9，所以点数是9。 
2. A.一杂五(5点)及 B.一斧头(11点)的组合，二牌点数之和为5+11=16，个位数为6，所以点数是6。 
3. A.一梅花(10点)及 B.一红头(10点)的组合，二牌点数之和为10+10=20，个位数为0，所以点数是0。 
真实情况四只牌共可有三种不同摆牌方法组合，AB-CD / AC-BD / AD-BC
例: 以四只牌组合为例:

1. A 及 B (0), C 及 D (0) 　前 0 点、后 0 点

2. A 及 C (5), B 及 D (5) 　前 5 点、后 5 点

3. A 及 D (3), B 及 C (7) 　前 3 点、后 7 点
因大牌九是每人四张牌，分为前后大小两组，分别与庄家比牌，全胜全败为胜负，一胜一败为和局。

庄家及玩家自定前和后两组牌点数最高最有利组合搭配，有对牌或称对子当然最好。若全局手上四张牌搭配较弱时，策略上可顾意分配其中一组较强，保守搭配一胜一败，宁和不输。

以图中为例，若以赌场法则 (House way)方法摆牌，应以组合 2. A 及 C (5), B 及 D (5) /　前5点、后5点 之搭配比起其他两种组合较为可取。以点数大小及牌顺序排列为: 前(小)一组 零霖5点、后(大)一组 板凳5点。 
单独丁三牌及二四牌可以作 3点 或 6点 

大细鸡可变比较特别，战前是没有。清代小说有关推牌九大细鸡亦无变，于近几十年香港澳门才盛行，可叫作广东牌九例；亦因中国禁赌多年，只有澳门能与国际其他赌场接轨，促进中西赌博文化交流带出牌九(Pai Gow)及骰宝(Sic Bo)，所以如今世界各大赌场包括美洲，欧洲，澳洲及东南亚等牌九赌台上，大细鸡可变亦已成惯例。
例: 歌词来自郑君绵的「赌仔自叹」

「拎林六　长衫六　高脚七　仲有一只大头六　二三更　瓜老衬　输到我木」
若获得一张零霖六牌及一张二四牌(大头六/大鸡)，原本视为6+6=12，只有2点，全局前后组合为 6+6=12 及 6+7=13，即前2点、后3点，所谓二三更的意思。
如今赌场例丁三牌(丁鸡/细鸡)、二四牌(大头六/大鸡)可以视情况作3点 或 6点，全局前后组合为 6+7=13 及 6+可以视为3=9，现即前3点、后9点。
单独丁三牌及二四牌可以作 3点 或 6点令全局配牌方法更多，取胜或保守策略选择更广，整体游戏趣味性更丰富。
